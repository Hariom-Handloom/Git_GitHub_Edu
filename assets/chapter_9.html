üåç What is a Remote Repository?

A remote repository is a version of your project that is hosted on the internet or a network.

  • Platforms like GitHub, GitLab, or Bitbucket host these remotes.
  • They allow you to collaborate with others, share code, and safely back up your work.
  • Common remote name: origin

üîó Connecting to a Remote

To connect a local Git project to a remote repository (e.g., a GitHub repo):

git remote add origin https://github.com/username/repo-name.git

origin is the alias name for the remote URL.


You can verify:

git remote -v

üß≤ Cloning a Remote Repository

To copy a remote repository to your local machine:

git clone https://github.com/username/repo-name.git

This:

  • Downloads the code
  • Creates a local folder
  • Sets up origin remote by default

üì§ Pushing Changes to Remote

Once your code is committed locally, use:

git push origin branch-name

Example: Push main branch

git push origin main

If it's your first push, you may need to set the upstream:

git push -u origin main

This links your local branch with the remote.

üì• Pulling Changes from Remote

To download changes from the remote repository:

git pull origin main

This:

  • Fetches latest changes
  • Merges them into your local branch

üîÑ Fetch vs Pull

Command Description
git fetch Gets updates from remote but doesn‚Äôt merge
git pull Fetches + merges automatically

Use git fetch if you want to review changes before merging.

üö® Common Push Error: Rejected Push

! [rejected]        main -> main (non-fast-forward)

This means someone else pushed to the remote before you.

Solution:

git pull origin main --rebase

Then push again:

git push origin main

üõ†Ô∏è Changing or Removing Remotes

View remotes:

git remote -v

Change remote URL:

git remote set-url origin NEW_URL

Remove a remote:

git remote remove origin

üåê Working with Multiple Remotes

You can add more than one remote, for example:

git remote add backup https://gitlab.com/user/repo.git

Push to another remote:

git push backup main
