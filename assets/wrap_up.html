🚧 Final Project Ideas (Apply What You’ve Learned)

Choose any of the following based on your interest:

🧑‍💻 Solo Developer Project

  • Build a small static website or web app
  • Use a main and dev branch
  • Implement multiple feature/ branches for different parts
  • Track tasks using GitHub Issues
  • Document the process in a README
  • Tag your releases and publish using GitHub Pages

🤝 Open-Source Contribution Challenge

  • Find a GitHub project with “good first issue” tags
  • Fork the repo, make a fix, and open a PR
  • Follow community guidelines and learn from code reviews
  • Keep your fork in sync and track progress
