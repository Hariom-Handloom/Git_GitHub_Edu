Git is a distributed version control system that tracks changes in your files and allows multiple people to collaborate on the same project, even offline.

Git lets you:

  • Save versions of your project over time
  • View or go back to earlier versions
  • Work with others without overwriting each other's code
  • Sync your work to remote servers like GitHub

ğŸ—ï¸ How Does Git Work?

Git works like a timeline of snapshots.

Every time you save (called a commit), Git stores a snapshot of the current state of your project. If something breaks, you can travel back to a previous snapshot.
