üßë Step 4: Configure Git (First-Time Setup)

To personalize your commits, Git needs your name and email. Run the following commands:

git config --global user.name "Your Name"
git config --global user.email "youremail@example.com"

Replace the values with your actual name and email. These details will be attached to every commit you make.
You can check if it‚Äôs set correctly by running:

git config --global --list

It will show:

user.name=Your Name
user.email=youremail@example.com

üîç Where Git Stores Config

Git has three levels of configuration:

Level Command Applies To
System --system All users on the machine
Global --global Current user
Local (default) Current project only (stored in .git/config)

Most of the time, you‚Äôll use --global or no flag (local).
